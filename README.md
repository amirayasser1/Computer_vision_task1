"# Computer_vision_task1" 

## Task 1 GUI

PyQt5 + OpenCV GUI for:
- Load RGB and gray images.
- Add Uniform/Gaussian/Salt & Pepper noise with parameters.
- Apply Average/Gaussian/Median filters with kernel control.
- Edge detection: Sobel, Roberts, Prewitt (custom kernels + convolution) and Canny (OpenCV).

### Run

1) Install requirements:

```
pip install -r requirements.txt
```

2) Start the app:

```
python main.py
```
